hx-target and hx-get Examples

A CSS query selector of the element to target.

Content to replace via CSS Selector.

<button hx-get="/getpage" hx-target="#cssSelectorExample" class="btn btn-primary">CSS Selector Target</button>
<div id="cssSelectorExample">
  <p>Content to replace via CSS Selector.</p>
</div>

this which indicates that the element that the hx-target attribute is on is the target.

Content will be replaced in the button itself 
<button hx-get="/getpage" hx-target="this" class="btn btn-secondary"><b>this</b> Target</button>
<em>Content will be replaced in the button itself</em>

closest <CSS selector> which will find the closest ancestor element or itself, that matches the given CSS selector (e.g. closest tr will target the closest table row to the element).

Content to replace via closest selector.

<div class="closest-target">
  <button hx-get="/getpage" hx-target="closest .closest-target" class="btn btn-success">closest Target</button>
  <p>Content to replace via closest selector.</p>
</div>
find <CSS selector> which will find the first child descendant element that matches the given CSS selector.
find Target

Content to replace via find selector.

<div hx-get="/getpage" hx-target="find .find-internal">
  <div class="btn btn-danger btn sm">find Target</div>
  <div class="find-internal">
    <p>Content to replace via find selector.</p>
  </div>
</div>
next <CSS selector> which will scan the DOM forward for the first element that matches the given CSS selector. (e.g. next .error will target the closest following sibling element with error class)

Content to replace via next selector.

<button hx-get="/getpage" hx-target="next .next-target" class="btn btn-warning">next Target</button>
<div class="next-target">
  <p>Content to replace via next selector.</p>
</div>
previous <CSS selector> which will scan the DOM backwards for the first element that matches the given CSS selector. (e.g. previous .error will target the closest previous sibling with error class)

Content to replace via previous selector.

<div class="previous-target">
  <p>Content to replace via previous selector.</p>
</div>
<button hx-get="/getpage" hx-target="previous .previous-target" class="btn btn-info">previous Target</button>
